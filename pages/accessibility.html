דלג לתוכן הראשי

הצהרת נגישות

תאריך עדכון אחרון: פברואר 2026

1. מחויבותנו לנגישות

אתר "מטפל לכל אחד" (בית המטפלים) מחויב להנגשת התוכן והשירותים שלו לכלל האוכלוסייה, לרבות אנשים עם מוגבלויות. אנו פועלים בהתאם לתקן הישראלי (ת"י 5568) ולהנחיות WCAG 2.1 ברמת AA.

2. פעולות הנגשה שבוצעו

במסגרת מאמצי ההנגשה, האתר כולל בין היתר:

  • תמיכה מלאה בניווט באמצעות מקלדת (Tab, Enter, Escape)
  • מחווני מיקוד (Focus indicators) ברורים על כל אלמנט אינטראקטיבי
  • קישור "דלג לתוכן הראשי" (Skip to content) בראש כל עמוד
  • תגיות alt לתמונות ו-aria-label לכפתורים עם אייקונים בלבד
  • תמיכה בכיוון טקסט מימין לשמאל (RTL) בכל הדפים
  • מבנה סמנטי תקין (header, nav, main, footer) לקריאת מסך
  • ניגודיות צבעים העומדת בדרישות WCAG AA
  • תמיכה במצב כהה (Dark Mode) בפורטל הלמידה
  • אזורי נגיעה מינימליים בגודל 44x44 פיקסלים במכשירים ניידים
  • כיבוד הגדרת "צמצום תנועה" (prefers-reduced-motion) של המשתמש

3. טכנולוגיות בשימוש

האתר נבנה תוך שימוש בטכנולוגיות הבאות, התומכות בנגישות:

  • HTML5 סמנטי
  • CSS3 עם משתני נושא (CSS Custom Properties)
  • JavaScript עם תמיכה ב-ARIA roles ו-attributes
  • Tailwind CSS עם תמיכה מובנית בנגישות

4. דפדפנים וטכנולוגיות מסייעות

האתר נבדק ותואם לעבודה עם:

  • Google Chrome (גרסה אחרונה)
  • Mozilla Firefox (גרסה אחרונה)
  • Microsoft Edge (גרסה אחרונה)
  • Safari (גרסה אחרונה)
  • קורא מסך NVDA

5. מגבלות ידועות

למרות מאמצינו, ייתכנו אזורים באתר שטרם הונגשו באופן מלא:

  • תוכן וידאו — חלק מסרטוני הקורסים עשויים שלא לכלול כתוביות מלאות
  • מסמכי PDF — חלק מהמסמכים להורדה עשויים שלא להיות מונגשים לקוראי מסך

אנו עובדים באופן שוטף לשפר ולהנגיש את כל תכני האתר.
